# 마크다운(Markdown) 이란?  

- ### 마크업 언어의 일종으로 일반 텍스트 문서의 양식을 편집하는 문법입니다.  
- ### 텍스트를 HTML로 변환하는 언어 입니다.  

# 마크다운(Markdowm) 사용법  
  
## **1 문장& 문단**

- ### 문장 끝에 공백 두칸은 줄 바꿈 기능을 합니다.
  (문장 끝 띄어쓰기 2칸 => \<br>)
  - 공백 없을 경우   
  
     \<!-- markdown --> 

     Hello World(띄어쓰기 없음)  
     HI MD  
    
     \<!--HTML-->     
     
     \<p>Hello World HI MD\</p>
